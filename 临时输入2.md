ContainerNo	箱号	大文本	双击汇总多行装箱单的箱号[CTNNO]/封条号[SEALNO]	暂空

GoodsName	品名	大文本

PkgsNum	件数	int	可为空，不为空时校验大于零的整数		
PkgsType	包装类型	string 32	简单字典PackType的code

MeasureMent	体积(CBM)	float	三位小数，可为空，不为空时校验大于零

MBLIssueNumber	正本张数	int	可为空，不为空时校验大于零的整数

MBLIssueDate	签发时间	datetime			
MBLIssuePlace	签发地点	string 32			
mblgoodsmode	放货方式	string 32	简单字典GoodsReleaseMode的code		
Remark	备注	大文本			
MBLRemark	提单附页	大文本
